# TRAFFIC LIGHTS

A simple LED pattern generator written in pure C++ for the Arduino Uno.  
This project simulates a basic traffic light sequence using three LEDs.

## Hardware
- Arduino Uno
- LEDs (3x)
- 220Ω resistors (3x)
- Breadboard (1x)
- Jumper Wires (4x)

## Wiring
LED1 → Pin 13  
LED2 → Pin 12  
LED3 → Pin 11  

## Code
Written in standard C++ with `#include <Arduino.h>` so GitHub detects the language properly.
